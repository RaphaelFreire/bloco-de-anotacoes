# Dicas do Terminal

Alguns Comandos:

- `cd` (Para navegar entre as pastas pelo terminal)

```
 cd nomeDaPasta
```

Exemplo de como sair de uma pasta:

```
cd ..
```

- `mkdir` (Para criar uma pasta)

Exemplo de como criar uma pasta:

```
mkdir nome-da-pasta
```

# Dicas do Git

- `git commit -m "mensagem"`
- `git status`
- `git log`
